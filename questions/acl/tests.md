# Тесты: ACL (Access Control Lists)

## Вопросы с множественным выбором

1. **В каком порядке маршрутизатор обрабатывает правила в ACL?**  
   A. В случайном порядке.  
   B. Снизу вверх.  
   C. Сверху вниз, до первого совпадения.  
   D. Всегда проверяет сначала deny.  
   **Правильный ответ: C**

2. **Какой из диапазонов номеров обычно используется для Standard ACLs?**  
   A. 100–199  
   B. 1–99  
   C. 2000–2699  
   D. 500–599  
   **Правильный ответ: B**

3. **Если вы хотите фильтровать трафик по номеру порта (например, порт 23 для Telnet), какой тип ACL вы должны использовать?**  
   A. Standard ACL  
   B. Named Standard ACL  
   C. Extended ACL  
   D. Port ACL  
   **Правильный ответ: C**

4. **Что произойдет, если пакет не соответствует ни одному из правил permit в ACL?**  
   A. Пакет будет отправлен.  
   B. Пакет будет отброшен (заблокирован).  
   C. Пакет будет перенаправлен.  
   D. Маршрутизатор спросит администратора.  
   **Правильный ответ: B**

5. **Какое ключевое слово в ACL используется для указания одного конкретного IP-адреса, например, 192.168.1.1?**  
   A. any  
   B. host  
   C. single  
   D. exact  
   **Правильный ответ: B**

6. **Какой номер диапазона используется для Extended ACL?**  
   A. 1-99  
   B. 100-199  
   C. 1300-1999  
   D. 2000-2699  
   **Правильный ответ: B**

7. **Что означает wildcard маска 0.0.0.255?**  
   A. Разрешает все IP-адреса  
   B. Блокирует последний октет  
   C. Разрешает последний октет  
   D. Игнорирует последний октет  
   **Правильный ответ: D**

## Инструкции

Каждый вопрос имеет 4 варианта ответа. Выберите один правильный. Правильные ответы отмечены в конце каждого вопроса для самопроверки.

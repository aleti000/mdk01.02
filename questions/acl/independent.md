# Вопросы для самостоятельной работы: ACL (Access Control Lists)

## Задания

1. **Напишите команду Standard ACL (с номером 1), которая разрешит абсолютно весь трафик.**

2. **Напишите команду, которая применит ACL с номером 10 к интерфейсу GigabitEthernet 0/0 для фильтрации входящего трафика.**

3. **Какое ключевое слово используется в ACL для обозначения любого IP-адреса (0.0.0.0 с маской 255.255.255.255)?**

4. **Напишите команду Extended ACL (с номером 100), которая разрешит любой TCP-трафик между любыми двумя хостами.**

5. **Какую команду нужно использовать в режиме привилегированного EXEC для просмотра всех настроенных ACL на устройстве?**

## Рекомендации

Студенты должны предоставить точные команды Cisco IOS, объяснения их работы и примеры применения в реальных сценариях сетевой безопасности.

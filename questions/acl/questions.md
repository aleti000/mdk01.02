# Вопросы для устных ответов: ACL (Access Control Lists)

## Вопросы

1. **Какова основная функция ACL (Access Control List)?**

2. **Назовите два основных типа ACL, используемых в Cisco-устройствах.**

3. **Что такое "Implicit Deny" (Неявный запрет) в ACL?**

4. **Какой тип ACL (Standard или Extended) может фильтровать трафик только на основе IP-адреса источника?**

5. **Какие два направления (direction) можно указать при применении ACL к интерфейсу маршрутизатора?**

6. **Что такое Named ACL и чем они отличаются от Numbered ACL?**

7. **Как работает Time-based ACL и для чего они используются?**

8. **Что такое Object Groups в ACL и как они упрощают управление?**

## Рекомендации для ответов

Студенты должны демонстрировать понимание принципов работы ACL, уметь объяснять различия между типами списков доступа и приводить примеры практического применения.

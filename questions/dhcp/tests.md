# Тесты: DHCP

## Вопросы с множественным выбором

1. **Что делает DHCP-сервер?**  
   A) Управляет маршрутизацией  
   B) Раздает IP-адреса клиентам автоматически  
   C) Обеспечивает безопасность сети  
   D) Контролирует доступ к интернету  
   **Правильный ответ: B**

2. **Какой порядок этапов при выдаче IP клиенту?**  
   A) Request -> Discover -> Offer -> Ack  
   B) Offer -> Discover -> Ack -> Request  
   C) Discover -> Offer -> Request -> Ack  
   D) Request -> Ack -> Discover -> Offer  
   **Правильный ответ: C**

3. **Для чего используется команда ip helper-address?**  
   A) Чтобы маршрутизатор стал DHCP-сервером  
   B) Чтобы пересылать DHCP-запросы в другую сеть  
   C) Чтобы задать шлюз по умолчанию  
   D) Чтобы задать адрес DNS  
   **Правильный ответ: B**

4. **Как исключить диапазон адресов из DHCP-пула?**  
   A) ip dhcp exclude range  
   B) ip dhcp pool exclude  
   C) ip dhcp excluded-address  
   D) ip dhcp address remove  
   **Правильный ответ: C**

5. **Что означает параметр default-router в DHCP-пуле?**  
   A) IP-адрес DHCP-сервера  
   B) IP шлюза по умолчанию для клиентов  
   C) Адрес DNS-сервера  
   D) Маска подсети  
   **Правильный ответ: B**

6. **Какой порт использует DHCP-сервер для отправки предложений?**  
   A) 67 UDP  
   B) 68 UDP  
   C) 67 TCP  
   D) 68 TCP  
   **Правильный ответ: A**

7. **Что такое lease time в DHCP?**  
   A) Время аренды IP-адреса  
   B) Время отклика сервера  
   C) Время обновления таблицы  
   D) Время ожидания клиента  
   **Правильный ответ: A**

## Инструкции

Каждый вопрос имеет 4 варианта ответа. Выберите один правильный. Правильные ответы отмечены в конце каждого вопроса для самопроверки.

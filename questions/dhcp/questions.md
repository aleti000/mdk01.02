# Вопросы для устных ответов: DHCP

## Вопросы

1. **Что такое DHCP и для чего он используется в сетях?**

2. **Опиши процесс получения IP-адреса клиентом (DORA-процесс).**

3. **Что такое DHCP-пул и какие параметры в него входят?**

4. **Какие основные параметры кроме IP-адреса может выдать DHCP-сервер клиенту?**

5. **В чем преимущество динамической раздачи IP-адресов по сравнению со статической настройкой?**

6. **Что такое DHCP relay agent и когда он используется?**

7. **Какие DHCP options можно настроить и для чего они нужны?**

8. **Что такое DHCP snooping и как оно повышает безопасность сети?**

## Рекомендации для ответов

Студенты должны уметь объяснять принципы работы DHCP, описывать процесс получения адреса и приводить примеры практического применения в корпоративных сетях.

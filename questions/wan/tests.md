# Тесты: WAN

## Вопросы с множественным выбором

1. **Какой тип WAN-соединения предоставляет выделенный, постоянный канал между двумя точками?**  
   a) коммутация пакетов (Frame Relay)  
   b) арендованные линии (T1/E1)  
   c) коммутация каналов (ISDN)  
   g) DSL  
   **Правильный ответ: b**

2. **Какой протокол канального уровня обеспечивает функцию аутентификации и управления качеством канала?**  
   a) HDLS  
   b) Frame Relay  
   c) PPP  
   g) Ethernet  
   **Правильный ответ: c**

3. **Какая технология использует ячейки фиксированного размера 53 байта для передачи данных?**  
   a) Frame Relay  
   b) ISDN  
   c) ATM  
   g) WAN  
   **Правильный ответ: c**

4. **Какой тип DSL обеспечивает одинаковую скорость передачи данных на приём и отправку?**  
   a) ADSL  
   b) VDSL  
   c) SDSL  
   g) ADSL2+  
   **Правильный ответ: c**

5. **Какое устройство агрегирует подключение множества абонентов DSL в провайдерской сети?**  
   a) DSLAM  
   b) Модем  
   c) Коммутатор  
   g) Маршрутизатор  
   **Правильный ответ: a**

6. **Какая технология WAN обеспечивает гарантированную пропускную способность?**  
   a) Интернет  
   b) Арендованные линии  
   c) Dial-up  
   d) Беспроводная сеть  
   **Правильный ответ: b**

7. **Что означает DLCI в Frame Relay?**  
   a) Data Link Connection Identifier  
   b) Digital Line Control Interface  
   c) Data Link Control Identifier  
   d) Digital Link Connection Interface  
   **Правильный ответ: a**

## Инструкции

Каждый вопрос имеет 4 варианта ответа. Выберите один правильный. Правильные ответы отмечены в конце каждого вопроса для самопроверки.

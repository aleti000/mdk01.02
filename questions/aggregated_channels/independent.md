# Вопросы для самостоятельной работы: Агрегированные каналы

## Задания

1. **Нарисуйте схему топологии сети с агрегированным каналом между двумя коммутаторами. Покажите все компоненты: физические порты, логический канал, DR и BDR (если применимо).**

2. **Рассчитайте общую пропускную способность агрегированного канала, состоящего из 4 гигабитных Ethernet-линков. Объясните, как распределяется нагрузка в различных режимах балансировки.**

3. **Проанализируйте следующую конфигурацию EtherChannel на коммутаторе Cisco:**
   ```
   interface Port-channel 1
    switchport mode trunk
    switchport trunk allowed vlan 10,20,30
   interface GigabitEthernet 0/1
    channel-group 1 mode active
   interface GigabitEthernet 0/2
    channel-group 1 mode active
   ```
   Определите возможные проблемы и предложите улучшения.

4. **Спроектируйте агрегированный канал для файлового сервера в корпоративной сети. Определите количество портов, режим агрегации и обоснуйте свой выбор.**

5. **Опишите процесс установления LACP-сессии между двумя устройствами. Нарисуйте диаграмму состояний и объясните каждый шаг.**

6. **Сравните статическую и динамическую агрегацию каналов. Создайте таблицу преимуществ и недостатков каждого подхода.**

7. **Решите задачу: В сети используется агрегированный канал из 3 портов по 1 Гбит/с каждый. Рассчитайте минимальную и максимальную пропускную способность для одного TCP-соединения в различных режимах.**

8. **Проанализируйте сценарий отказа одного линка в агрегированном канале. Опишите, как система реагирует и какие механизмы обеспечивают непрерывность работы.**

## Рекомендации

Студенты должны предоставить письменные ответы, схемы и расчеты. Поощряется использование реальных примеров конфигураций и анализ практических сценариев.

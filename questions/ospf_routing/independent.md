# Вопросы для самостоятельной работы: Маршрутизация OSPF

## Задания

1. **Нарисуйте топологию OSPF-сети с backbone областью (area 0) и двумя дополнительными областями (area 1 и area 2). Покажите ABR, маршрутизаторы и связи между ними.**

2. **Рассчитайте стоимости маршрутов в OSPF-сети. Даны интерфейсы: GigabitEthernet (cost 1), FastEthernet (cost 10), Serial (cost 64). Постройте таблицу маршрутизации для простой топологии из 3 маршрутизаторов.**

3. **Проанализируйте конфигурацию OSPF на маршрутизаторе Cisco:**
   ```
   router ospf 1
    network 192.168.1.0 0.0.0.255 area 0
    network 10.0.0.0 0.255.255.255 area 1
   ```
   Определите, какие сети будут анонсироваться и в какие области.

4. **Спроектируйте OSPF-сеть для кампуса университета с 3 зданиями. Определите количество областей, типы областей и разместите ABR.**

5. **Опишите процесс установления соседства OSPF между двумя маршрутизаторами. Нарисуйте диаграмму состояний от Down до Full и объясните каждый переход.**

6. **Создайте таблицу сравнения типов LSA в OSPF. Для каждого типа укажите: название, генерируется кем, описание содержимого, область распространения.**

7. **Решите задачу: В OSPF-сети с reference bandwidth 100 Мбит/с рассчитайте стоимость интерфейсов: 10 Мбит/с, 100 Мбит/с, 1 Гбит/с, 10 Гбит/с.**

8. **Проанализируйте сценарий разделения backbone области. Предложите решение с использованием виртуальных ссылок и нарисуйте схему.**

## Рекомендации

Студенты должны предоставить схемы, расчеты и письменные объяснения. Рекомендуется использовать реальные примеры команд конфигурации и анализ выходных данных команд show.

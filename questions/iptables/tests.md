# Тесты: iptables

## Вопросы с множественным выбором

1. **В какой таблице по умолчанию работают правила, если при создании правила таблица не указана явно?**  
   a) nat  
   b) mangle  
   c) filter  
   d) raw  
   **Правильный ответ: c**

2. **Какая команда используется для добавления нового правила в начало цепочки INPUT?**  
   a) iptables -I INPUT  
   b) iptables -A INPUT  
   c) iptables --insert INPUT  
   d) iptables --prepend INPUT  
   **Правильный ответ: a**

3. **Какое действие (TARGET) правило выполнит, если пакет соответствует всем критериям этого правила?**  
   a) Обработка пакета переходит к следующему правилу в цепочке.  
   b) Пакет немедленно передается приложению.  
   c) Пакет принимается (ACCEPT) или отбрасывается (DROP/REJECT) в соответствии с решением этого правила.  
   d) Цепочка возвращается к первому правилу.  
   **Правильный ответ: c**

4. **Для настройки проброса порта (DNAT) правило следует добавлять в цепочку...**  
   a) INPUT таблицы filter  
   b) PREROUTING таблицы nat  
   c) OUTPUT таблицы filter  
   d) POSTROUTING таблицы nat  
   **Правильный ответ: b**

5. **В чем основное различие между действиями DROP и REJECT?**  
   a) DROP логирует пакет, а REJECT — нет.  
   b) REJECT отправляет обратно пакет с ошибкой, а DROP просто игнорирует пакет.  
   c) DROP работает только в цепочке FORWARD, а REJECT — только в INPUT.  
   d) Разницы нет, это синонимы.  
   **Правильный ответ: b**

## Инструкции

Каждый вопрос имеет 4 варианта ответа. Выберите один правильный. Правильные ответы отмечены в конце каждого вопроса для самопроверки.
